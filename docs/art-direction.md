# 美术方向 v0.1

## 核心风格

- 精致欧洲旅行绘本，手绘动画电影质感，不仿制《旅行青蛙》的具体画风。
- 温暖但不幼稚：自然纸张肌理、克制的细节、电影式光线。
- 主色：苔藓绿、奶油白、勃艮第红、旧金色；不同城市有自己的辅色。
- 所有叙事插画均通过 Imagegen 生成，真实照片只作为地点、构图和氛围参考。
- 画面不直接生成文字，标题和时间由网页覆盖，保证可读性。

## 小狗角色（已选定）

- 选择：第一版角色设定页右上方案。
- 外形：小型奶油白梗犬，略微蓬乱的硬质毛发，直立三角耳，深棕色温柔眼睛，黑色鼻头。
- 体态：四肢短而结实，正常幼犬比例，不做夸张大头或过度幼化。
- 固定服装：苔藓绿短款旅行外套、棕色皮革斜挎小包。
- 性格观感：萌、好奇、略显认真；不是用户本人，而是替用户携带回忆的旅行伙伴。
- 角色参考：`assets/references/dog-character-options-v1.png` 右上角。生成场景时只参考右上角色，禁止混入其他三只狗的特征。

下一步先生成该角色的正面、侧面、背面与表情设定，再生成所有场景；每张场景图均引用同一角色设定。

## 资产规格

- 主场景：横向 3:2，网页裁切时保留角色和关键地标。
- 明信片：横向 4:3，不生成边框和文字。
- 物品卡：方形构图，纯色背景，后续可做透明抠图。
- 每张图必须保存对应 prompt、地点 ID、参考照片和版本号。
